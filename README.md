# CVApp
This Android mobile application is a comprehensive CV showcase built entirely in Kotlin using Jetpack Compose. It provides a user-friendly interface for viewing and editing personal and professional details. The app incorporates the ViewModel, Compose, and Navigation Compose libraries to create a seamless and responsive user experience.

# Installation
   1. Clone this repository:
      ```
      git clone https://github.com/robert-nganga/CVApp.git
      ```
   1. Open the project in Android Studio.
   1. Build and run the app on your preferred device or emulator.
